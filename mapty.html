Mapty

JavaScript

A Map application that uses geolocation feature and allow user to log their excercises (Running or Cycling) by clicking the location on the map. The data will be stored in the browser's local storage.

Rules and Features :

1.When user open the browser, leaflet's map will be loaded with stored datas.

2.User may chose which excercises they want to log, running or cycling. After chosing the excercise, the input units may change.

3.Data will be stored and displayed on map after user press 'Enter'.

4.Implement mapping feature that when user click on a specific data, the map will move to the corresponding location.
