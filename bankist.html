Bankist App

JavaScript

A simple banking application that requires and checks user login informations. After user logged in successfully , application will display all the transaction histories. Also, it will display the total deposit, total withdrawal and current balance of the current user. Bankist also provides basic banking features such as request a loan from the bank, transfer money between accounts and close the current account.

Rules and Features :

1.Takes user's account and pin numbers, then check if inputs match the database.

2.If user logged in successfully, Bankist will display the user-interface. Also start the logout timer.

3.Add Sorting button that sort and unsort the transactions.

4. User may request a loan but the bank will only allow if the current user has at least 1 deposit that is 10% of the requested amount. If loan is approved, the amount will be added to current balance.

5. User may transfer money between accounts with 2 conditions : the account must existed in database and amount must be positive.

6. User may close their account with correct account and pin numbers.
