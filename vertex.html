Vertex App

JavaScript

A Java program implement with graphics and Jframe which allow user to draw vertices on the JPanel by clicking on the panel area. After drawing the vertices, user may add any edge between the vertices. Also has other features such as moving the vertices, remove any vertex or edge.

Rules and Features :

1.Add one or more vertices by clicking on the Jpanel.

2.Add edge by clicking on 2 vertices only if a edge does not existed between the chosen vertices.

3.Remove one or more vertices on the panel. If an edge is attached to the chosen vertex, the edge will be removed also.

4.Remove one or more edges on the panel.

5.Move vertex around the panel, if there is an edge attached to the vertex , the edge's length and direction will be updated also.
